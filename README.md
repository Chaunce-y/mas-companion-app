# Margaritaville at Sea Companion App

A mobile-first cruise companion app concept designed to enhance the onboard experience for guests.

## Features
- Trip dashboard with countdown
- Dining discovery
- Daily schedule planning
- Ship navigation
- Account management

## Tech Stack
- React Native (Expo)
- TypeScript
- Expo Router

## Purpose
This project reimagines the digital experience for Margaritaville at Sea, providing a modern, mobile-native interface for cruise guests.
